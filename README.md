##Project Overview
MoodSync is an intelligent AI-powered system designed to detect the emotional state of a user and play suitable music accordingly. The project uses Artificial Intelligence, Machine Learning and Deep Learning techniques to analyse the user's mood through facial expressions. The main goal of MoodSync is to improve mental wellness, relaxation, productivity and entertainment by synchronising music with the user's emotional state.

##Objectives
- Detect user mood accurately using AI models
- Recommend songs based on detected emotional inputs
- Improve user experience through personal music playback
- Support emotional wellness using music therapy concepts

##Technologies Used
- Backend: Node.js, Express.js
- Frontend: HTML, CSS, JavaScript
- Machine Learning: Python (CNN Model)
- Libraries: OpenCV, TensorFlow, Keras, NumPy, Pandas

##Features
- Emotion detection using CNN model 
- Automatic music playback based on detected mood 
- Real time user input processing
- Personalised and adaptive user experience 
- Potential integration with mental health and entertainment platforms
