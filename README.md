# MoodSync
##Description

MoodSync is an emotionally aware music playback system which analyses user input to detect emotions and play context appropriate music. This creates an empathetic and interactive user experience. This can be utilised in mental health websites and entertainment platforms, making mood sink versatile. Built using a Convolutional Neural Network (CNN) model to detect user emotions.

##Features

- Emotion detection using CNN model 
- Automatic music playback based on detected mood 
- Real time user input processing
- Personalised and adaptive user experience 
- Potential integration with mental health and entertainment platforms 

##Tech Stack

- Backend: Node.js, Express.js
- Frontend: HTML, CSS, JavaScript
- Machine Learning: Python (CNN Model)
- Libraries: OpenCV, TensorFlow, Keras, NumPy, Pandas

##How To Run

1. Clone the repository 
2. Navigate to the project folder 
3. Install dependencies using npm install 
4. Start the server using node server.js 
5. Open the application in your browser
